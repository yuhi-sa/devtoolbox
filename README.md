# DevToolBox 🛠️

85種類以上の開発者向け無料オンラインツール集。JSON整形、Base64変換、正規表現テスター、CSSジェネレーター、JWT解析など、日常の開発作業を効率化します。

🌐 **[https://yuhi-sa.github.io/devtoolbox/](https://yuhi-sa.github.io/devtoolbox/)** | [English](https://yuhi-sa.github.io/devtoolbox/en/)

## ✨ 特徴

- 🆓 **完全無料・登録不要** - ブラウザだけで即利用
- 🔒 **プライバシー重視** - データはすべてブラウザ内で処理、サーバーへの送信なし
- 📱 **PWA対応** - オフラインでも利用可能、ホーム画面に追加できる
- 🌏 **多言語対応** - 日本語 / English
- 🌙 **ダークモード** - ライト/ダークテーマ切替対応
- ⌨️ **コマンドパレット** - `Cmd+K` でツールを素早く検索

## 🚀 ローカル開発

```bash
git clone https://github.com/yuhi-sa/devtoolbox.git
cd devtoolbox
# 任意のHTTPサーバーで起動
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。

## 🛠️ 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- **ホスティング**: GitHub Pages
- **データ処理**: すべてクライアントサイドで完結（サーバー不要）
- **PWA**: Service Worker によるオフライン対応
