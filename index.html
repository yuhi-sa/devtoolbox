DevToolBox

開発者の日常作業を効率化する、無料オンラインツール集

49 種類のツールをすべて無料で利用可能

🔍

該当するツールが見つかりません。別のキーワードで検索してください。

フォーマット・整形

JSON整形・検証 おすすめ

JSONデータをワンクリックで見やすく整形・圧縮・検証。構文エラーも分かりやすく表示します。

SQLフォーマッター

SQL文を見やすく整形。キーワードの大文字化やインデント付与を自動で行います。

XMLフォーマッター

XMLデータを見やすく整形・圧縮。構文検証やインデント設定にも対応。

Markdownプレビュー

Markdownをリアルタイムでプレビュー。見出し、リスト、テーブル、コードブロックなどに対応。

SVGオプティマイザー NEW

SVGコードを最適化・軽量化。コメント除去、メタデータ削除、パス簡略化に対応。

CSSミニファイ/整形 NEW

CSSコードの圧縮（ミニファイ）と整形（ビューティファイ）。サイズ削減率も表示。

JSミニファイ/整形 NEW

JavaScriptコードの圧縮と整形。コメント除去、空白削除でファイルサイズを削減。

HTMLミニファイ/整形 NEW

HTMLコードの圧縮と整形。コメント除去、空白正規化で軽量化。

エンコード・デコード

Base64エンコード/デコード おすすめ

テキストをBase64にエンコード・デコード。日本語（マルチバイト文字）にも正確に対応。

URLエンコード/デコード

URLエンコード・デコードをオンラインで実行。日本語URLの変換にも対応。

HTMLエンティティ変換

HTMLの特殊文字をエンティティにエンコード・デコード。名前付き・数値エンティティに対応。

画像Base64変換

画像ファイルをBase64データURLに変換。ドラッグ&ドロップ対応。逆変換も可能。

エスケープ変換

バックスラッシュエスケープの変換・復元。\\n、\\t、\\r等の特殊文字に対応。

文字列エスケープ変換 NEW

JavaScript、Python、Java、C等の各言語向け文字列エスケープ・アンエスケープ変換。

テキスト処理

文字数カウンター おすすめ

文字数・行数・バイト数をリアルタイムでカウント。スペース含む/除くの切替にも対応。

正規表現テスター おすすめ

正規表現パターンをリアルタイムでテスト。マッチ結果のハイライト表示やキャプチャグループの確認が可能。

正規表現チートシート

正規表現の構文リファレンス。基本パターン、量指定子、文字クラス、先読み/後読みを一覧表示。

テキスト差分比較 おすすめ

2つのテキストを比較し、追加・削除行をハイライト表示。コードレビューや文書の変更確認に。

テキストケース変換

camelCase、snake_case、PascalCase、kebab-case等に変換。10種類以上の変換に対応。

HTML→Markdown変換

HTMLコードをMarkdown形式に変換。見出し、リスト、テーブル、コードブロックに対応。

Lorem Ipsumジェネレーター

ダミーテキストを段落・文・単語数で生成。日本語ダミーテキストにも対応。

スラッグ生成

テキストからURL用スラッグを生成。区切り文字や最大長のカスタマイズに対応。

文字数・単語カウンター NEW

文字数、単語数、文数、段落数、読了時間をリアルタイム計測。キーワード密度分析も対応。

テキスト→HTML変換 NEW

プレーンテキストをHTMLに変換。URL自動リンク化、改行変換、特殊文字エスケープに対応。

データ変換

YAML⇔JSON変換

YAMLとJSONの相互変換。設定ファイルの形式変更に便利。

CSV⇔JSON変換

CSVとJSONの相互変換。区切り文字の選択やヘッダー行の設定に対応。

JSON→CSV変換

JSONデータをCSV形式に変換・ダウンロード。ネストされたオブジェクトの展開にも対応。

TOML⇔JSON変換

TOMLとJSONの相互変換。Cargo.toml等の設定ファイル変換に便利。

JSON→TypeScript変換

JSONデータからTypeScriptのinterface/typeを自動生成。ネスト構造にも対応。

JSONパスファインダー

JSONをツリー表示し、クリックでJSONPathを取得。パスクエリのテストも可能。

JSONスキーマバリデーター NEW

JSONデータをJSONスキーマに対して検証。バリデーションエラーの詳細表示付き。

変換ツール

Unixタイムスタンプ変換

Unixタイムスタンプと日時を相互変換。現在時刻の取得やタイムゾーン変換にも対応。

カラーコンバーター

HEX、RGB、HSL間の相互変換。カラーピッカーとリアルタイムプレビュー付き。

進数変換ツール

2進数・8進数・10進数・16進数を相互変換。BigInt対応で大きな数値も処理可能。

バイト単位変換

B、KB、MB、GB、TB、PBを相互変換。10進（1000）と2進（1024）の両モード対応。

画像圧縮 おすすめ

ブラウザ上で画像を圧縮・リサイズ。JPEG、PNG、WebP出力に対応。品質調整可能。

アスペクト比計算機 NEW

アスペクト比の計算と寸法変換。16:9、4:3、1:1など一般的な比率のプリセット付き。

生成ツール

ハッシュ生成 おすすめ

MD5、SHA-1、SHA-256、SHA-512のハッシュ値をリアルタイムで生成。

UUID生成

UUID v4を暗号学的に安全な乱数で生成。一括生成（最大100個）にも対応。

パスワード生成

安全なパスワードを生成。文字数・文字種のカスタマイズと強度メーター付き。

QRコード生成

テキストやURLからQRコードを生成。サイズ・色のカスタマイズやPNGダウンロードに対応。

カラーパレット生成

ベースカラーから調和のとれた配色パレットを生成。補色、類似色、トライアドに対応。

プレースホルダー画像生成 NEW

任意サイズのプレースホルダー画像を生成。背景色・テキスト色のカスタマイズとダウンロード対応。

CSS・デザイン

CSSグラデーション生成 おすすめ

線形・放射グラデーションをビジュアルに作成。カラーストップの追加やCSS出力に対応。

Box Shadow生成

box-shadowをスライダーで直感的に設定。複数シャドウやinsetに対応。

Flexboxプレイグラウンド

Flexboxプロパティをインタラクティブに操作。ライブプレビューでレイアウトを確認。

Border Radius生成

4つの角のborder-radiusをスライダーで設定。リンク/アンリンク切替に対応。

CSSグリッドジェネレーター NEW

CSS Gridレイアウトをビジュアルに構築。行・列・ギャップ設定とCSSコード出力に対応。

開発支援

JWTデコーダー

JWTトークンをデコードしてHeader・Payload・Signatureを表示。有効期限チェック付き。

JWTジェネレーター NEW

カスタムヘッダー・ペイロードからJWTトークンを生成。HS256アルゴリズム対応。

Cron式パーサー

Cron式を日本語で解説し、次の実行予定時刻を表示。プリセット付きで簡単入力。

Crontabジェネレーター NEW

GUIでCron式を直感的に作成。プリセットや次回実行時刻の確認にも対応。

Chmod計算機

ファイルパーミッションを数値（755）とシンボル（rwxr-xr-x）で相互変換。

HTTPステータスコード

全HTTPステータスコードの検索可能なリファレンス。カテゴリ別フィルター付き。

.gitignore生成

言語・フレームワーク別のテンプレートから.gitignoreを生成。複数選択・カスタマイズ可能。

メタタグ生成

SEO・OGP・Twitter Card用のメタタグをフォーム入力で生成。文字数チェック付き。

OGPプレビュー

OGP設定のプレビュー。Facebook・Twitter・Slack・LINE風のカード表示で確認。

OGPテスター NEW

OGPメタタグの検証・テスト。推奨画像サイズや文字数チェック、各SNSプレビュー付き。

Robots.txtジェネレーター NEW

robots.txtをGUIで簡単生成。クローラー別のルール設定やAIボットブロックプリセット付き。
