Privacy Policy

Last updated: February 11, 2026

DevToolBox (hereinafter "this site") values your privacy. This Privacy Policy explains how data is handled on this site.

1. Data Collection

All tools provided by this site (JSON formatting, character counting, Base64 encoding, URL encoding, Unix timestamp conversion, etc.) process data entirely within your browser.

Data entered by users into the tools is never sent to any external server. All processing is performed by JavaScript within the browser, and input data exists only in browser memory and is discarded when you leave the page.

2. Use of Local Storage

This site uses browser local storage for the following purposes:

  • Saving theme preferences (light mode / dark mode)

Data stored in local storage does not contain any personal information. It can be deleted at any time through your browser settings.

3. Cookies

This site itself does not use its own cookies. However, the following services may use cookies.

Advertising Networks

This site may use third-party advertising networks in the future. Advertising networks may use cookies to display ads based on user interests. Please refer to each advertising network's privacy policy for management of these cookies.

Analytics

This site may use analytics tools to understand site usage. Analytics tools use cookies to collect access information, but no personally identifiable information is included. Data is used solely for site improvement purposes.

4. External Links

This site may contain links to external websites. We are not responsible for the privacy practices of external sites. Please review the privacy policy of linked sites.

5. Children's Privacy

This site does not intentionally collect personal information from children under 13 years of age.

6. Changes to Privacy Policy

This site may update this Privacy Policy as necessary. Any significant changes will be communicated on this page.

7. Contact

If you have any questions about this Privacy Policy, please contact us via the GitHub repository Issues.
