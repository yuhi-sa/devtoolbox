JSONвҶ’TypeScriptеӨүжҸӣ

JSONгғҮгғјгӮҝгӮ’иІјгӮҠд»ҳгҒ‘гҒҰгҖҒTypeScriptгҒ®гӮӨгғігӮҝгғјгғ•гӮ§гғјгӮ№гӮ’иҮӘеӢ•з”ҹжҲҗгҒ—гҒҫгҒҷгҖӮ

JSONвҶ’TypeScriptеӨүжҸӣгғ„гғјгғ«гҒ®дҪҝгҒ„ж–№

е·ҰеҒҙгҒ®гғҶгӮӯгӮ№гғҲгӮЁгғӘгӮўгҒ«JSONгғҮгғјгӮҝгӮ’иІјгӮҠд»ҳгҒ‘гҖҒгҖҢеӨүжҸӣгҖҚгғңгӮҝгғігӮ’гӮҜгғӘгғғгӮҜгҒҷгӮӢгҒЁгҖҒеҸіеҒҙгҒ«TypeScriptгҒ®гӮӨгғігӮҝгғјгғ•гӮ§гғјгӮ№е®ҡзҫ©гҒҢз”ҹжҲҗгҒ•гӮҢгҒҫгҒҷгҖӮгғҚгӮ№гғҲгҒ—гҒҹгӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ«гҒҜеҖӢеҲҘгҒ®гӮӨгғігӮҝгғјгғ•гӮ§гғјгӮ№гҒҢиҮӘеӢ•з”ҹжҲҗгҒ•гӮҢгҒҫгҒҷгҖӮй…ҚеҲ—еҶ…гҒ®гӮӘгғ–гӮёгӮ§гӮҜгғҲеһӢгӮӮжҺЁи«–гҒ•гӮҢгҒҫгҒҷгҖӮгҖҢexportгҖҚгӮӘгғ—гӮ·гғ§гғігӮ’гӮӘгғігҒ«гҒҷгӮӢгҒЁгҖҒеҗ„гӮӨгғігӮҝгғјгғ•гӮ§гғјгӮ№гҒ«exportгӮӯгғјгғҜгғјгғүгҒҢд»ҳеҠ гҒ•гӮҢгҒҫгҒҷгҖӮ

еҜҫеҝңгҒҷгӮӢеһӢеӨүжҸӣ

д»ҘдёӢгҒ®еһӢеӨүжҸӣгҒ«еҜҫеҝңгҒ—гҒҰгҒ„гҒҫгҒҷпјҡ

  • ж–Үеӯ—еҲ— вҶ’ string
  • ж•°еҖӨ вҶ’ number
  • зңҹеҒҪеҖӨ вҶ’ boolean
  • null вҶ’ null
  • й…ҚеҲ— вҶ’ иҰҒзҙ еһӢгӮ’жҺЁи«–пјҲstring[], number[] гҒӘгҒ©пјү
  • гғҚгӮ№гғҲгҒ—гҒҹгӮӘгғ–гӮёгӮ§гӮҜгғҲ вҶ’ еҲҘгҒ®гӮӨгғігӮҝгғјгғ•гӮ§гғјгӮ№гҒЁгҒ—гҒҰз”ҹжҲҗ
  • ж··еҗҲеһӢгҒ®й…ҚеҲ— вҶ’ гғҰгғӢгӮӘгғіеһӢпјҲ(string | number)[]пјү
