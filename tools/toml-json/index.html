TOML-JSON変換ツール

TOMLとJSONを相互変換できるオンラインツールです。外部ライブラリ不要で動作します。

TOML-JSON変換ツールの使い方

このツールは、TOML（Tom's Obvious, Minimal Language）とJSON（JavaScript Object Notation）を相互に変換できるオンラインツールです。「TOML → JSON」タブでTOMLをJSONに変換し、「JSON → TOML」タブでJSONをTOMLに変換できます。設定ファイルの形式変換やデータフォーマットの確認に便利です。

TOMLとは？

TOML（Tom's Obvious, Minimal Language）は、設定ファイル向けに設計されたデータシリアライゼーション言語です。人間が読みやすく書きやすいことを目的としており、Cargo.toml（Rust）、pyproject.toml（Python）、Hugo（Go）などで広く使用されています。キー・値ペア、テーブル（セクション）、配列、インラインテーブルなどの構文をサポートし、コメント（#）も使用可能です。JSONと比較して、コメントが書ける点やよりシンプルな構文が特徴です。

よくある質問（FAQ）

Q. どのようなTOML構文に対応していますか？

A. キー・値ペア（ベアキー・クォートキー）、基本文字列・リテラル文字列、整数・浮動小数点数・真偽値、配列、テーブル、インラインテーブル、コメントに対応しています。

Q. 入力したデータは安全ですか？

A. はい、安全です。すべての処理はブラウザ内で完結し、入力データがサーバーに送信されることは一切ありません。
