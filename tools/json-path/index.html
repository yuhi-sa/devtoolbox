JSONгғ‘гӮ№гғ•гӮЎгӮӨгғігғҖгғј

JSONгғҮгғјгӮҝгӮ’гғ„гғӘгғјиЎЁзӨәгҒ—гҖҒеҖӨгӮ’гӮҜгғӘгғғгӮҜгҒ—гҒҰJSONPathгӮ’еҸ–еҫ—гҒ§гҒҚгҒҫгҒҷгҖӮJSONPathгӮҜгӮЁгғӘгӮӮе®ҹиЎҢеҸҜиғҪгҒ§гҒҷгҖӮ

JSONгғ‘гӮ№гғ•гӮЎгӮӨгғігғҖгғјгҒ®дҪҝгҒ„ж–№

гҒ“гҒ®гғ„гғјгғ«гҒҜгҖҒJSONгғҮгғјгӮҝгӮ’иҰ–иҰҡзҡ„гҒӘгғ„гғӘгғјж§ӢйҖ гҒ§иЎЁзӨәгҒ—гҖҒд»»ж„ҸгҒ®еҖӨгӮ’гӮҜгғӘгғғгӮҜгҒҷгӮӢгҒ“гҒЁгҒ§гҒқгҒ®JSONPathејҸгӮ’еҸ–еҫ—гҒ§гҒҚгҒҫгҒҷгҖӮгҒҫгҒҹгҖҒJSONPathгӮҜгӮЁгғӘгӮ’е…ҘеҠӣгҒ—гҒҰи©ІеҪ“гҒҷгӮӢеҖӨгӮ’жӨңзҙўгҒҷгӮӢгҒ“гҒЁгӮӮеҸҜиғҪгҒ§гҒҷгҖӮAPIгғ¬гӮ№гғқгғігӮ№гҒ®и§ЈжһҗгӮ„гғҮгғјгӮҝж§ӢйҖ гҒ®зўәиӘҚгҒ«дҫҝеҲ©гҒ§гҒҷгҖӮ

JSONPathгҒЁгҒҜпјҹ

JSONPathгҒҜгҖҒJSONж–ҮжӣёеҶ…гҒ®зү№е®ҡгҒ®иҰҒзҙ гӮ’еҸӮз…§гҒҷгӮӢгҒҹгӮҒгҒ®гӮҜгӮЁгғӘиЁҖиӘһгҒ§гҒҷгҖӮXPathгҒҢXMLж–ҮжӣёгҒ«еҜҫгҒ—гҒҰдҪҝгӮҸгӮҢгӮӢгҒ®гҒЁеҗҢж§ҳгҒ«гҖҒJSONPathгҒҜJSONгғҮгғјгӮҝгҒ®зү№е®ҡгҒ®еҖӨгӮ„гғҺгғјгғүгӮ’жҢҮе®ҡгҒҷгӮӢгҒҹгӮҒгҒ«дҪҝз”ЁгҒ•гӮҢгҒҫгҒҷгҖӮгғ«гғјгғҲиҰҒзҙ гҒҜгҖҢ$гҖҚгҒ§иЎЁгҒ•гӮҢгҖҒгғүгғғгғҲиЁҳжі•пјҲ$.store.bookпјүгӮ„гғ–гғ©гӮұгғғгғҲиЁҳжі•пјҲ$['store']['book']пјүгҒ§гӮўгӮҜгӮ»гӮ№гҒ§гҒҚгҒҫгҒҷгҖӮй…ҚеҲ—гҒ®гӮӨгғігғҮгғғгӮҜгӮ№гӮўгӮҜгӮ»гӮ№пјҲ$[0]пјүгӮ„гғҜгӮӨгғ«гғүгӮ«гғјгғүпјҲ$.*гҖҒ$[*]пјүгҒ«гӮӮеҜҫеҝңгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

A. гғүгғғгғҲиЁҳжі•пјҲ$.keyпјүгҖҒгғ–гғ©гӮұгғғгғҲиЁҳжі•пјҲ$['key']пјүгҖҒй…ҚеҲ—гӮӨгғігғҮгғғгӮҜгӮ№пјҲ$[0]пјүгҖҒгғҜгӮӨгғ«гғүгӮ«гғјгғүпјҲ$.*гҖҒ$[*]пјүгҒ«еҜҫеҝңгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ
